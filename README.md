# Usage
To start the program enter following command in project root directory:
```
cargo run
```
The first move is made by crosses.\
\
Running program:
```
Turn X
         1 2 3
      1 | | | |
      2 | | | |
      3 | | | |
Enter the (x, y) (in range 1..3) for X, splitted with space
```
Enjoy the game
